# **App Name**: HearMeOut

## Core Features:

- Room Creation: Users can create private or public rooms for voice chatting and music streaming.
- Synchronized Audio Player: Play music from YouTube URLs, with playback synchronized for all users in the room using Firebase. Users can also upload local audio files which will be hosted in Firebase Storage.
- Audio Ducking: Automatically adjust music volume when users are speaking, ensuring voices are heard clearly.
- Voice Chat: Enable real-time voice communication between users within a room, prioritizing clear and low-latency audio using WebRTC.
- Overlay URL: Provide an overlay URL that displays information about who is speaking and the currently playing music, customizable for streaming.
- OAuth2 Authentication: Allow users to authenticate with Discord and Twitch accounts, managing authentication via Firebase.
- Moderation tool: Utilize the moderation tool to determine the overall sentiment of the ongoing conversations within a room, flagging potentially harmful or inappropriate content and alerting administrators as needed.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to create a calming and social atmosphere.
- Background color: Very light gray (#F5F5F5), almost white, for a clean and unobtrusive backdrop that allows content to stand out.
- Accent color: Pale blue (#ADD8E6) for interactive elements and highlights.
- Body and headline font: 'PT Sans', a humanist sans-serif that is both modern and welcoming, used for both headings and body text.
- Use simple, outlined icons for navigation and interactive elements.
- A clean, card-based layout with intuitive navigation to ensure a seamless user experience.
- Subtle transitions and animations to enhance user interaction without being distracting.
